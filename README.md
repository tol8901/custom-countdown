# custom-countdown
In this project you can set a countdown title and a date, after that you'll see a countdown (which is possible to reset). The data of the countdown is saved in the Local Storage, it means, after reload the page, the countdown save prevoius data.
# The project is available
 https://tol8901.github.io/custom-countdown/
